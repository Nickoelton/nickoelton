Personaliza la terminal de Windows

Bienvenida o bienvenido a la nueva experiencia del Terminal de Windows. En esta entrada podrás encontrar el paso a paso de cómo instalar y configurar la terminal. Además te enseñaré a como colocar un fondo transparente a tu Terminal, con la finalidad que se vea súper futurista.

Luego de un breve preámbulo, empezamos con la instalación:

  • Primero, descargamos la aplicación de Windows Terminal desde el Microsoft Store aquí.

Luego de completar la instalación, nos quedará un acceso rápido desde el Menú de Inicio de Windows.

Al iniciar la aplicación podremos visualizar nuestra nueva terminal.

Para poder personalizarla, primero debemos aperturar la barra de mas opciones en la parte superior y luego seleccionar "Configuración". Esto nos aperturará una ventana con la cual podremos ir cambiando la apariencia, iteracción y acciones de nuestra Terminal.

Como nosotros vamos a configurar el tema acrílico o transparente a nuestra Terminal, nos iremos a la última opción de la barra lateral llamada "Abrir Archivo JSON". Para poder abrirlo necesitarás de un editor de código instalado, en mi caso usaré Visual Studio Code.

En nuestro editor de código, vamos a buscar la sección JSON que encapsule los "Profiles", y en la propiedad "Defaults" será donde incrustemos nuestro código para personalizar de color acrílico o transparente nuestra Terminal.

A continuación, te dejo el código listo.

Copiar el código 

                                
        "defaults":
        {
            "useAcrylic": true,
            "acrylicOpacity": 0.1
        },

Por lo que una vez seteado, quedará así:

Luego de guardar los cambios en Visual Studio Code, regresarás a la terminal y ¡SORPRESA! tendrás una Terminal de Windows con una apariencia muy moderna y fresca, lista para usarse.
