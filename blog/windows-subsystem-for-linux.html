Conociendo Windows Subsystem for Linux

Hola estimada o estimado lector. Como dice el título de esta entrada, vamos empezar a dar nuestros primeros pasos en Ubuntu conociendo Windows Subsystem for Linux (WSL) o como se dice en español el Subsistema de Windows para Linux. Para lograr ello, es importante que conozcamos a detalle de que se trata y cual es la forma correcta de configurarlo.

Empecemos definiendo que es WSL, el cual es un entorno dentro de Windows que permite a los usuarios o desarrolladores ejecutar distribuciones de GNU/Linux, las cuales pueden ser escogidas desde el Microsoft Store (Entre ellas se encuentra Ubuntu, SUSE, Kali, entre otras).

Para iniciar con la instalación y configuración, debemos cumplir con el siguiente requisito: La máquina que utlizaremos debe ejecutar Windows 10 version 2004 (Compilación 19041) o superior.

Lo siguiente es ejecutar la siguiente linea de comando desde PowerShell (No te olvides de ejecutar PowerShell como Administrador).

Copiar el código 

                                
    wsl --install

Esperamos la descarga... y una vez finalizado nos aparecerá un mensaje que diga "La operación se realizó correctamente. Los cambios se aplicarán cuando se reinicie el sistema".

A continuación, debemos ejecutar la siguiente linea de comando desde PowerShell, el cual habilitará las características necesarias para ejecutar WSL.

Copiar el código 

                                
    dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

Una ves completado, debemos indicarle a WSL que versión utilizará. En este caso usaremos la mas reciente, la cual es WSL2.

Para ello, nos toca ejecutar la siguiente linea de comando desde PowerShell.

Copiar el código 

                                
    wsl --set-default-version-2

Una vez finalizado, podremos salir de PowerShell con el comando "Exit".

Acto seguido, nos dirigimos al Microsoft Store y buscamos instalar el Sistema Operativo de preferencia, que para esta demostración utilizaremos Ubuntu 20.04 LTS (Su versión mas reciente y estable). Puedes ingresar directamente desde aquí para descargar Ubuntu 20.04 LTS.
